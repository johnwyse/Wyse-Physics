Acceleration

Acceleration = (final velocity - initial velocity)/(time)
a = (vf-vi)/t

source: https://www.youtube.com/watch?v=4dCrkp8qgLU&ab_channel=ProfessorDaveExplains

Variable Definition Symbol Unit
Acceleration The rate at which velocity changes a meters per second per second (m/s/s)
Initial Velocity How fast an object is moving at the beginning of an event vi meters per second (m/s)
Final Velocity How fast an object is moving at the end of an event vf meters per second (m/s)
Time The duration of a given event t seconds (s)

Example Calculation

What was the acceleration of a car that sped up from 19 m/s to 34 m/s in 5 seconds?

a = ? m/s/s
vi = 19 m/s
vf = 34 m/s
t = 5 s

a = (vf-vi)/t

a = (34-19)/5
a = 15/5
a = 3 m/s/s
